# Student Details Management System
A Spring Boot REST API that manages student records using a layered architecture.

## How to Run
1. Open the project in IntelliJ IDEA.
2. Run `StudetDetailsApplication.java`.
3. Visit `http://localhost:8080/api/students` to see the JSON output.

## Tech Stack
* Java 17+
* Spring Boot
* Spring Data JPA
* H2 In-memory Database
